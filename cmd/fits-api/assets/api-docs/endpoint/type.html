Type

Look up observation type information.

Query Index:

  • Type - List all observation types.

Type

List all observation types.

Method: GET
URI
/type
Accept
application/json;version=1

Query Parameters

Response Properties

description
Type description e.g., displacement from initial position
name
Type name e.g., east
typeID
A type identifier for observations e.g., e.
unit
Type unit e.g., mm.
