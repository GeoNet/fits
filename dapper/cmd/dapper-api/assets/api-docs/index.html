GeoNet API

Welcome to the Dapper API.

The GeoNet project makes all its data and images freely available. Please ensure you have read and understood our Data Policy and Disclaimer before using any of these services.

The DAPPER API provides access to records and associated meta data for a given domain.

Endpoints

The following endpoints are available:

  • /data - Look up record data.
  • /meta - Look up record metadata.

Formatting Output JSON

We use the jq command for JSON pretty printing etc. A curl command might look like:

curl "http://...API-QUERY..." | jq .

Compression

The response for a query can be compressed. If your client can handle a compressed response then the reduced download size is a great benifit. Gzip compression is supported. You can request a compressed response by including gzip in your Accept-Encoding header.

Bugs

The code that provide these services is available at https://github.com/GeoNet/fits If you believe you have found a bug please raise an issue or pull request there. Alternatively contact us detailing the issue.
